
# 📄 MobiCar — Conversion Rate Analysis
Product analytics project analyzing user funnel conversion in a car-sharing mobile app (MobiCar). The study identifies key drop-off points. Includes funnel analysis, conversion rate tracking over time, and insights into potential product improvements.

## 📌 Overview

Analysis of user funnel conversion in MobiCar (car-sharing mobile app) over 8 weeks.
Goal: investigate a 15% drop in Install → First Trip conversion.

## 📊 Approach
- Built weekly funnel conversion tracking in Power BI
- Analyzed step-by-step conversion changes
- Identified bottlenecks across the user journey

## 📉 Key Finding
- The drop in overall conversion is driven by a sharp decline in Registration → Search (Week 7: 0.71 → 0.49).
- Other funnel stages remained stable or showed minor fluctuations.

## 🧠 Conclusion
The issue is localized in the search entry stage, likely linked to recent changes in the search functionality, leading to reduced user engagement early in the product journey.

## 💡 Hypotheses
- degradation of search user experience after recent updates
- decrease in relevance of search results
- technical issues in the search functionality (load speed, errors, empty results)
- reduced availability of cars in search results

## 🔍 Next Steps
Analyze impact of search update (before/after comparison, cohort analysis)
Segment by platform and geography
Check search usability and technical performance metrics

## 📊 Output
Power BI dashboard includes:
- Overall funnel conversion trend (in the output folder)
- Step-by-step funnel breakdown
